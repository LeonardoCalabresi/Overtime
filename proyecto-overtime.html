Dashboard de Estadísticas
Overtime Basquet

Dashboard interactivo de estadísticas para una liga de básquet amateur argentina, construido con scraping, limpieza de datos y visualización en Tableau.

Descripción del proyecto

Overtime Basquet es una liga de básquet amateur argentina. Este proyecto construye un pipeline completo de datos: desde la extracción automatizada del sitio web de la liga hasta un dashboard interactivo en Tableau que permite explorar estadísticas de jugadores, equipos y partidos.

El dashboard incluye rankings dinámicos por métrica (puntos, rebotes, asistencias, etc.), filtros por categoría y torneo, estadísticas por partido, y un sistema de antiranking con premios informales para los resultados más llamativos de cada fecha.

Principales desafíos

1
Extracción de datos

El sitio de la liga tiene arbitrariedades en su estructura y no dispone de una API ni base de datos pública. Se utilizó Selenium para navegar dinámicamente y BeautifulSoup para parsear el HTML.

2
Limpieza y deduplicación

Los nombres de jugadores presentan variaciones mínimas de escritura entre partidos, generando duplicados. La normalización de nombres es un trabajo en progreso continuo con Pandas.

3
Rankings dinámicos en Tableau

Tableau tiene limitaciones para construir rankings que respondan a múltiples filtros simultáneos. Se requirieron cálculos de tabla y campos calculados avanzados para lograr el comportamiento deseado.

4
Imágenes de jugadores y equipos

Las imágenes en el sitio fuente no están asociadas fijamente a cada jugador o equipo, sino que se repiten por partido. Fue necesario un mapeo adicional para asignar correctamente cada imagen.

Estado y próximos pasos

El dashboard está activo y se actualiza con cada fecha de la liga. La limpieza de nombres de jugadores continúa siendo el principal trabajo pendiente. Como próximo paso se planea automatizar la actualización de datos con un script programado.
